Fortinet FortiOS/FortiProxy/FortiPAM/FortiSwitchManager 格式字符串漏洞

漏洞信息

漏洞名称： Fortinet FortiOS/FortiProxy/FortiPAM/FortiSwitchManager 格式字符串漏洞

漏洞编号：

  • CVE： CVE-2024-23113

漏洞类型： 格式字符串

漏洞等级： 严重

漏洞描述： Fortinet FortiOS fgfmd守护进程及相关Fortinet产品中存在一个格式字符串漏洞。此漏洞允许远程未经认证的攻击者通过向FGFM服务发送特制请求来执行任意代码或命令。漏洞由fgfmd守护进程中对外部控制的格式字符串的不当处理引起，可在无需认证的情况下被利用。成功利用可能导致系统完全被攻陷、数据泄露、横向移动及对网络基础设施的持久访问。

产品厂商： Fortinet

产品名称： FortiOS, FortiProxy, FortiPAM, FortiSwitchManager

影响版本： FortiOS v7.4.0 - v7.4.2, v7.2.0 - v7.2.6, v7.0.0 - v7.0.13; FortiProxy v7.4.0 - v7.4.2, v7.2.0 - v7.2.8, v7.0.0 - v7.0.14; FortiPAM v1.2.0, v1.1.0 - v1.1.2, v1.0.0 - v1.0.3; FortiSwitchManager v7.2.0 - v7.2.3, v7.0.0 - v7.0.3
