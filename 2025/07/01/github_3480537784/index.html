Alibaba Nacos 未授权账户创建漏洞

漏洞信息

漏洞名称： Alibaba Nacos 未授权账户创建漏洞

漏洞类型： 未授权访问

漏洞等级： 高危

漏洞描述： Nacos在默认配置中使用固定的JWT令牌密钥来认证用户。由于Nacos是一个开源项目，密钥是公开的，因此未经授权的攻击者可以使用这个固定的密钥伪造任何用户身份登录Nacos，以管理和操作后台接口功能。

产品厂商： Alibaba

产品名称： Nacos

搜索语法： title:”Nacos”
