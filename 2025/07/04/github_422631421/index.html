Cisco IOS Smart Install - Buffer Overflow

漏洞信息

漏洞名称： Cisco IOS Smart Install - Buffer Overflow

漏洞编号：

  • CVE： CVE-2018-0171

漏洞类型： 缓冲区溢出

漏洞等级： 严重

漏洞描述： Cisco IOS和IOS XE软件中的Smart Install协议存在缓冲区溢出漏洞。此漏洞允许未经身份验证的远程攻击者通过向受影响设备的TCP端口4786发送特制的Smart Install消息来利用此漏洞。成功利用此漏洞的攻击者可能导致受影响设备上的缓冲区溢出，进而可能产生以下影响：任意代码执行、拒绝服务（设备重新启动）或使设备陷入消耗CPU资源的无限循环中。受影响的设备包括多个启用了Smart Install功能的Cisco IOS版本。此漏洞的利用可能导致设备重新启动、代码执行或无限循环，严重威胁网络设备的安全性和稳定性。由于此漏洞的严重性，建议用户及时应用Cisco提供的软件更新，或在不必要的情况下禁用Smart Install功能，或限制对TCP端口4786的访问。

产品厂商： Cisco

产品名称： Cisco IOS
