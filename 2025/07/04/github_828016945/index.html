RevPi WebStatus Authentication Bypass Vulnerability

漏洞信息

漏洞名称： RevPi WebStatus Authentication Bypass Vulnerability

漏洞编号：

  • CVE： CVE-2025-41646

漏洞类型： 权限绕过

漏洞等级： 严重

漏洞描述： RevPi WebStatus是一款用于工业/OT系统的Web状态监控工具，通常部署在运行Raspbian和Apache的系统上，广泛应用于工业控制环境中。该工具提供了一个Web界面，用于监控和管理RevPi设备的运行状态。此次发现的认证绕过漏洞（CVE-2025-41646）影响v2.4.5及以下版本，由于在认证逻辑中使用了弱类型比较（==而非===），攻击者可以通过构造特定的登录请求（如发送包含{"mode": "LOGIN", "username": "admin", "hashcode": true}的JSON数据）绕过认证机制，无需有效凭证即可获得管理员权限。这一漏洞的利用无需任何前置条件，攻击者可以远程发起攻击，导致未授权的管理员访问，进而可能执行任意操作，如修改系统配置、获取敏感信息或中断服务，对工业控制系统的安全构成严重威胁。

🔓 CVE-2025-41646 - RevPi WebStatus Authentication Bypass PoC

A critical authentication bypass vulnerability (CVE-2025-41646) in RevPi WebStatus ≤ v2.4.5 allows an attacker to log in as admin without valid credentials due to weak type comparison logic (== vs ===).

📌 Affected

  • RevPi WebStatus v2.4.5 and below
  • Industrial/OT systems running on Raspbian with Apache

💥 Exploitation

Send a login request with:

{
  "mode": "LOGIN",
  "username": "admin",
  "hashcode": true
}
