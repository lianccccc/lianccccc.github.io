vBulletin replaceAdTemplate Remote Code Execution

漏洞名称： vBulletin replaceAdTemplate Remote Code Execution

漏洞编号：

  • CVE： 2025-48827, 2025-48828

漏洞类型： 命令执行

漏洞等级： 高危

漏洞描述： 该漏洞存在于vBulletin的AJAX API处理程序和模板渲染系统中，影响版本5.0.0至6.0.3。攻击者可以通过ajax/api/ad/replaceAdTemplate端点利用PHP反射API的不当使用，结合PHP 8.1+的变化，调用受保护的控制器方法。具体而言，它针对vB_Api_Ad::replaceAdTemplate()方法注入包含vb:if条件的模板，该条件评估攻击者提供的PHP代码，使用”system”($POST[])构造。恶意模板随后通过第二个未经认证的请求到ajax/render/ad执行。成功利用可导致任意命令执行，无需认证。

产品厂商： vBulletin

产品名称： vBulletin

影响版本： 5.0.0 <= version <= 6.0.3
