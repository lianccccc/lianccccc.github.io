Microsoft Windows Cross Device Service Elevation of Privilege Vulnerability

漏洞名称： Microsoft Windows Cross Device Service Elevation of Privilege Vulnerability

漏洞编号：

  • CVE： CVE-2025-24076

漏洞类型： 权限提升

漏洞等级： 高危

漏洞描述： Windows Cross Device Service中存在一个权限提升漏洞，允许低权限的本地攻击者通过覆盖关键DLL文件（CrossDevice.Streaming.Source.dll）来执行恶意代码，从而获得SYSTEM权限。此漏洞影响Windows 11多个版本、Windows Server 2025和Windows Server 2022 23H2（Server Core安装）。

产品厂商： Microsoft

产品名称： Windows Cross Device Service

影响版本： Windows 11 Versions 24H2, 23H2, 22H2 (x64 and ARM64), Windows Server 2025, Windows Server 2022 23H2 (Server Core installation)

来源： https://github.com/mbanyamer/CVE-2025-24076

类型： CVE-2025:github search
