Apache HTTP Server mod_proxy_uwsgi 信息泄露与远程代码执行漏洞

漏洞信息

漏洞名称： Apache HTTP Server mod_proxy_uwsgi 信息泄露与远程代码执行漏洞

漏洞编号：

  • CVE： CVE-2020-11984

漏洞类型： 信息泄露

漏洞等级： 严重

漏洞描述： Apache HTTP Server 2.4.32至2.4.44版本中的mod_proxy_uwsgi模块存在信息泄露及可能的远程代码执行漏洞，允许远程攻击者通过发送特制请求访问敏感信息并可能执行任意代码。

产品厂商： Apache

产品名称： HTTP Server

影响版本： 2.4.32 <= version <= 2.4.44

搜索语法： cpe:”cpe:2.3:a:apache:http_server”

来源： https://github.com/projectdiscovery/nuclei-templates/issues/12266

类型： projectdiscovery/nuclei-templates:github issues

来源概述

Description:

Apache HTTP Server 2.4.32 to 2.4.44 contains an info disclosure and possible remote code execution caused by a vulnerability in mod_proxy_uwsgi, letting remote attackers access sensitive information and potentially execute arbitrary code, exploit requires sending crafted requests.
