Motors 未授权权限提升漏洞

漏洞名称： Motors 未授权权限提升漏洞

漏洞编号：

  • CVE： CVE-2025-4322

漏洞类型： 权限提升

漏洞等级： 严重

漏洞描述： WordPress的Motors主题在5.6.67及之前的所有版本中存在权限提升漏洞，攻击者可以在未经身份验证的情况下通过更新任意用户密码（包括管理员密码）来接管账户，从而获得对账户的访问权限。

产品厂商： WordPress

产品名称： Motors theme for WordPress

影响版本： version <= 5.6.67

搜索语法： body=”/wp-content/themes/motors/style.css”
