Ivanti EPMM Authentication Bypass for Expression Language Remote Code Execution

漏洞名称： Ivanti EPMM Authentication Bypass for Expression Language Remote Code Execution

漏洞编号：

  • CVE： CVE-2025-4427, CVE-2025-4428

漏洞类型： 命令执行

漏洞等级： 严重

漏洞描述： 该模块利用了Ivanti EPMM中的一个未经验证的远程代码执行漏洞链，跟踪为CVE-2025-4427和CVE-2025-4428。一个认证缺陷允许未经认证访问管理员Web API端点，从而允许通过表达式语言注入执行代码。此模块在’tomcat’用户上下文中执行。此模块也适用于许多版本的MobileIron Core（重新命名为Ivanti EPMM）。

产品厂商： Ivanti

产品名称： Ivanti EPMM (formerly MobileIron Core)
