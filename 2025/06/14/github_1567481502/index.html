Cisco IOS Smart Install Buffer Overflow Vulnerability

漏洞信息

漏洞名称： Cisco IOS Smart Install Buffer Overflow Vulnerability

漏洞编号：

  • CVE： CVE-2018-0171

漏洞类型： 缓冲区溢出

漏洞等级： 严重

漏洞描述： Cisco IOS和IOS XE软件中的Smart Install功能存在缓冲区溢出漏洞，由于对Smart Install数据包数据的不当验证，未经认证的远程攻击者可以通过发送特制的Smart Install消息（TCP端口4786）触发设备重启、执行任意代码或导致无限循环。

产品厂商： Cisco

产品名称： Cisco IOS and IOS XE Software

搜索语法： cpe:”cpe:2.3:o:cisco:ios”

来源： https://github.com/projectdiscovery/nuclei-templates/issues/12325

类型： projectdiscovery/nuclei-templates:github issues

来源概述

Description:

Cisco IOS and IOS XE Software contain a buffer overflow caused by improper validation of Smart Install packet data, letting unauthenticated remote attackers trigger device reload, execute arbitrary code, or cause indefinite loops, exploit requires sending crafted Smart Install message on TCP port 4786.

Severity: Critical
