LabKey Server XML External Entity (XXE) Vulnerability

漏洞名称： LabKey Server XML External Entity (XXE) Vulnerability

漏洞编号：

  • CVE： CVE-2019-9757

漏洞类型： XML实体注入

漏洞等级： 高危

漏洞描述： LabKey Server 19.1.0版本中存在XML外部实体(XXE)注入漏洞。攻击者可以通过向visualization-exportImage.view或visualization-exportPDF.view端点发送包含XXE负载的SVG文件，从而读取本地文件。

产品厂商： labkey

产品名称： labkey_server

影响版本： 19.1.0

搜索语法： title=”LabKey”
