SureTriggers WordPress插件认证绕过漏洞

漏洞名称： SureTriggers WordPress插件认证绕过漏洞

漏洞编号：

  • CVE： CVE-2025-3102

漏洞类型： 权限绕过

漏洞等级： 高危

漏洞描述： SureTriggers WordPress插件存在一个认证绕过漏洞，允许未认证的攻击者创建管理员账户。这是由于在autheticate_user函数中对secret_key缺少空值检查。当插件安装并激活但未配置API密钥时，攻击者可以通过发送空的st_authorization头部绕过认证，触发保留给认证用户的操作。此漏洞影响版本≤1.0.78，严重性为高危(8.1)。

产品厂商： SureTriggers

产品名称： SureTriggers – All-in-One Automation Platform WordPress plugin

影响版本： ≤ 1.0.78

来源： https://github.com/B1ack4sh/CVE-2025-3102

类型： CVE-2025:github search
