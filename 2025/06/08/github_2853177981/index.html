Request-Baskets 服务器端请求伪造漏洞

漏洞名称： Request-Baskets 服务器端请求伪造漏洞

漏洞编号：

  • CVE： CVE-2023-27163

漏洞类型： 服务器端请求伪造

漏洞等级： 高危

漏洞描述： CVE-2023-27163是Request-Baskets中的一个服务器端请求伪造(SSRF)漏洞。攻击者可以利用forward_url参数对本地主机端口(127.0.0.1)进行暴力扫描，以识别绑定到本地接口的服务。一旦发现，攻击者可以通过易受攻击的转发机制访问这些服务。
