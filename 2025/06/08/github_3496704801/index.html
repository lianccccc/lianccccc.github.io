Remote for Mac 未授权远程代码执行漏洞

漏洞名称： Remote for Mac 未授权远程代码执行漏洞

漏洞类型： 未授权访问

漏洞等级： 高危

漏洞描述： 此模块利用了Remote for Mac版本2025.7及之前版本中的未授权远程代码执行漏洞，通过/api/executeScript端点。当目标系统上禁用身份验证时，攻击者可以执行任意AppleScript命令，包括通过do shell script执行的shell命令。所有版本直至2025.7（包括补丁版本）均易受攻击。

产品名称： Remote for Mac

影响版本： version <= 2025.7
