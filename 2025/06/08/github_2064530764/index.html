Redmagic 9s pro Multiple Kernel Vulnerabilities

漏洞名称： Redmagic 9s pro Multiple Kernel Vulnerabilities

漏洞编号：

  • CVE： CVE-2025-21832, CVE-2025-21830, CVE-2025-21829, CVE-2025-21826

漏洞类型： 权限提升

漏洞等级： 高危

漏洞描述： 在Redmagic 9s pro（NX769J）设备上发现了多个已知漏洞（CVE-2025-21832、CVE-2025-21830、CVE-2025-21829和CVE-2025-21826），这些漏洞影响了Android内核和驱动程序的核心组件，尽管最近有安全更新，但这些漏洞仍然可被利用。这些漏洞包括ION内存分配器中的使用后释放漏洞、Binder驱动程序中的竞态条件、Adreno GPU驱动程序中的边界检查不当以及PMIC子系统中的权限执行错误。这些漏洞可以被利用来进行本地权限提升、绕过SELinux强制执行和应用沙箱、安装持久性恶意软件、访问或窃取私人数据以及通过PMIC漏洞禁用或损坏硬件组件。

产品厂商： Nubia/Redmagic
