来源概述

CVE-2025-33053 Simple Vulnerability Checker (PowerShell)

A lightweight PowerShell script to detect if a Windows system is potentially vulnerable to CVE-2025-33053 — a critical WebDAV-related Remote Code Execution (RCE) vulnerability.

🛡️ About CVE-2025-33053

This vulnerability allows attackers to execute malicious .LNK shortcut files that point to remote WebDAV shares, resulting in arbitrary code execution.

The exploit requires the following conditions to be true on the target system:

  1. WebClient service is present and running.
  2. UNC path handling via WebDAV is active.

This checker tests for both these real-world preconditions.

✅ What This Checker Does (Real-Time Verification)

Check Purpose
🔍 WebClient Service Status Confirms if WebDAV support is active on the system.
🔍 UNC Path Handling via Test-Path Checks if the system tries to resolve remote WebDAV UNC paths (core to CVE exploitation).

If both checks pass — the system is potentially exploitable via CVE-2025-33053.

🖥️ How to Run (Usage)

  1. Save the above PowerShell script as checker.ps1.

  2. Open PowerShell as Administrator.

  3. If script execution is restricted, temporarily allow running scripts:

  1. Run the script: .\CVE-2025-33053_checker.ps1

🔍 Why is this a Real-Time Checker?

✔️ WebClient Service Status is checked live — no assumptions.
✔️ UNC Path Handling uses actual OS calls (Test-Path) — which triggers real UNC resolution logic the same way a .LNK exploit would do.

✅ If UNC handling resolves (even for non-existent paths), this reflects the exact behavior that enables exploitation — thus this is not theoretical, but a direct simulation of CVE exploitation preconditions.

⚠️ Notes

This script does not exploit the vulnerability.
It is a safe, read-only checker for determining potential exposure.
Suitable for blue teams, sysadmins, vulnerability auditors.
Only for educational purposes. Do not run on systems you are not authorised to run.
