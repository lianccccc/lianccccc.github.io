Roundcube Post-Auth RCE via PHP Object Deserialization

漏洞名称： Roundcube Post-Auth RCE via PHP Object Deserialization

漏洞编号：

  • CVE： 2025-49113

漏洞类型： 反序列化

漏洞等级： 高危

漏洞描述： Roundcube Webmail在1.5.10之前和1.6.x在1.6.11之前的版本中，由于program/actions/settings/upload.php中的_from参数未经验证，导致经过身份验证的用户可以通过PHP对象反序列化执行远程代码。攻击者可以以Web服务器用户的身份执行任意系统命令。

产品厂商： Roundcube

产品名称： Roundcube Webmail

影响版本： 1.6.x < 1.6.11, < 1.5.10
