ZoneMinder SQL注入漏洞

漏洞名称： ZoneMinder SQL注入漏洞

漏洞编号：

  • CVE： CVE-2024-51482

漏洞类型： SQL注入

漏洞等级： 高危

漏洞描述： ZoneMinder是一个免费的开源闭路电视软件应用程序。ZoneMinder v1.37.* <= 1.37.64版本中的web/ajax/event.php函数存在布尔型SQL注入漏洞，此漏洞在1.37.65版本中已修复。攻击者可以利用此漏洞通过构造特定的SQL语句，获取数据库中的敏感信息，如用户名和密码。
