VMware VMX TOCTOU Race Condition Vulnerability

漏洞名称： VMware VMX TOCTOU Race Condition Vulnerability

漏洞编号：

  • CVE： CVE-2025-22224

漏洞类型： 权限提升

漏洞等级： 高危

漏洞描述： 此漏洞存在于VMware ESXi和Workstation中，由于VMX进程中的TOCTOU（Time-of-Check to Time-of-Use）竞争条件导致。攻击者可以利用此漏洞在主机上执行任意代码，从而提升权限。漏洞的严重性被评估为高危，CVSSv3评分为8.2。

产品厂商： VMware

产品名称： VMware ESXi, VMware Workstation

影响版本： version < 17.5

来源： https://github.com/lacelruz/CVE-2025-22224
