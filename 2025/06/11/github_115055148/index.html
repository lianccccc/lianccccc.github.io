漏洞信息

漏洞名称： Remote for Mac 2025.6 Unauthenticated UDP Keyboard RCE

漏洞类型： 命令执行

漏洞等级： 高危

漏洞描述： 此模块利用了Remote for Mac 2025.6中的一个未经身份验证的远程代码执行漏洞。当启用“允许未知设备”设置时，攻击者可以通过UDP数据包模拟键盘输入而无需身份验证。通过发送一系列按键，攻击者可以打开终端并执行任意shell命令，实现代码执行。
