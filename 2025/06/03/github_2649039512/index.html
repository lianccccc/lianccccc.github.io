Metasploit remove_host Module Hard-coded Hosts File Path Vulnerability

漏洞名称： Metasploit remove_host Module Hard-coded Hosts File Path Vulnerability

漏洞类型： 配置问题

漏洞等级： 中危

漏洞描述： Metasploit Framework中的remove_host模块存在硬编码的主机文件路径问题，当Windows系统未安装在C盘时，模块无法正确找到hosts文件，导致功能失效。建议使用环境变量SystemRoot或windir动态确定hosts文件的正确位置。

产品名称： Metasploit Framework
