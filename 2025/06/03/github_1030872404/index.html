Oracle WebLogic Server 远程代码执行漏洞

漏洞名称： Oracle WebLogic Server 远程代码执行漏洞

漏洞编号：

  • CVE： CVE-2020-2883

漏洞类型： 命令执行

漏洞等级： 严重

漏洞描述： Oracle WebLogic Server (10.3.6.0.0, 12.1.3.0.0, 12.2.1.3.0, 12.2.1.4.0) 存在远程代码执行漏洞，攻击者可以通过IIOP和T3协议未经认证访问服务器，从而控制服务器。此漏洞需要通过网络访问IIOP或T3协议来利用。

产品厂商： Oracle

产品名称： WebLogic Server

影响版本： 10.3.6.0.0, 12.1.3.0.0, 12.2.1.3.0, 12.2.1.4.0
