漏洞信息

漏洞名称： OpenNMS Horizon 认证远程代码执行漏洞

漏洞类型： 命令执行

漏洞等级： 高危

漏洞描述： OpenNMS Horizon存在一个认证远程代码执行漏洞，允许攻击者在认证后通过特定的HTTP请求执行任意命令。

产品厂商： OpenNMS

产品名称： OpenNMS Horizon

Steps to reproduce

How’d you do it?

  2. Show payloads returns no results.
  3. Trying to run the exploit with the exploit’s default configured payload (cmd/linux/http/x64/meterpreter/reverse_tcp) returns an incompatible payload error.
