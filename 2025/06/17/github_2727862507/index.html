Roundcube Webmail PHP对象反序列化漏洞

漏洞信息

漏洞名称： Roundcube Webmail PHP对象反序列化漏洞

漏洞编号：

  • CVE： CVE-2025-49113

漏洞类型： 反序列化

漏洞等级： 严重

漏洞描述： Roundcube Webmail在1.5.10之前和1.6.x在1.6.11之前的版本中，由于program/actions/settings/upload.php中的_from参数未经验证，导致认证用户可以通过PHP对象反序列化执行远程代码。

产品厂商： Roundcube

产品名称： Roundcube Webmail

影响版本： version < 1.5.10, version < 1.6.11

来源概述

CVE‑2025‑49113 – Post‑Auth Remote Code Execution in Roundcube via PHP Object Deserialization

Description:

Roundcube Webmail before 1.5.10 and 1.6.x before 1.6.11 allows remote code execution by authenticated users because the _from parameter in a URL is not validated in program/actions/settings/upload.php, leading to PHP Object Deserialization.
