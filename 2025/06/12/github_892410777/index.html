Windows Server 远程系统强制关机权限检查

漏洞信息

漏洞名称： Windows Server 远程系统强制关机权限检查

漏洞类型： 权限绕过

漏洞等级： 中危

漏洞描述： 验证“从远程系统强制关机”策略（SeRemoteShutdownPrivilege）是否仅授予管理员组（SID S-1-5-32-544）。未经授权的账户拥有此权限可能使攻击者能够远程关闭系统。如果管理员组以外的账户或组拥有此权限，可能会被利用执行未经授权的远程关机，导致服务中断或拒绝服务。

产品厂商： Microsoft

产品名称： Windows Server
