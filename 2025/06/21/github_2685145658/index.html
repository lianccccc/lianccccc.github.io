Microsoft SharePoint 远程代码执行漏洞

漏洞信息

漏洞名称： Microsoft SharePoint 远程代码执行漏洞

漏洞编号：

  • CVE： CVE-2019-0604

漏洞类型： 远程代码执行

漏洞等级： 严重

漏洞描述： Microsoft SharePoint存在一个远程代码执行漏洞，原因是未能检查应用程序包的源标记，允许远程攻击者执行任意代码。利用此漏洞需要发送恶意的应用程序包。
