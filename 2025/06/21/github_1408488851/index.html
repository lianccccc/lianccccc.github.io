Microsoft NET Framework 远程代码执行漏洞

漏洞信息

漏洞名称： Microsoft .NET Framework 远程代码执行漏洞

漏洞编号：

  • CVE： CVE-2020-0646

漏洞类型： 命令执行

漏洞等级： 严重

漏洞描述： Microsoft .NET Framework 存在一个远程代码执行漏洞，原因是输入验证不当，允许远程攻击者执行任意代码。利用此漏洞需要向系统发送恶意输入。

产品厂商： Microsoft

产品名称： .NET Framework

搜索语法： server: ms .net remoting

来源： https://github.com/projectdiscovery/nuclei-templates/issues/12210

类型： projectdiscovery/nuclei-templates:github issues
