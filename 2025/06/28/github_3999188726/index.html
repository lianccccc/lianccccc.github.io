ISPConfig language_editphp PHP代码注入

漏洞信息

漏洞名称： ISPConfig language_edit.php PHP代码注入

漏洞编号：

  • CVE： CVE-2023-46818

漏洞类型： 代码注入

漏洞等级： 高危

漏洞描述： ISPConfig的language_edit.php文件中存在PHP代码注入漏洞。当admin_allow_langedit设置启用时，经过身份验证的管理员可以通过语言编辑器界面注入任意PHP代码。该模块会自动检查所需的admin_allow_langedit权限是否启用，并尝试在禁用时启用它（需要具有系统配置访问权限的管理员凭据）。漏洞利用通过将PHP有效负载注入到语言文件中，然后在访问该文件时执行。有效负载使用PHP的file_put_contents函数进行base64编码和写入。

产品厂商： ISPConfig

产品名称： ISPConfig
