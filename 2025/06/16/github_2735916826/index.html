Zimbra UnRAR Path Traversal Vulnerability

漏洞信息

漏洞名称： Zimbra UnRAR Path Traversal Vulnerability

漏洞编号：

  • CVE： CVE-2023-30333

漏洞类型： 目录穿越

漏洞等级： 高危

漏洞描述： CVE-2023-30333影响Linux/UNIX系统上的UnRAR工具，版本低于6.12。攻击者可利用此漏洞进行路径遍历，将文件写入解压目录之外的任意位置，可能导致任意代码执行。此漏洞的CVSS v3.1评分为7.5（高危），无需特权或用户交互即可利用，且已被NVD、CISA、CERT-EU和SonarSource确认存在实际攻击案例。漏洞存在于UnRAR的IsRelativeSymlinkSafe()函数中，该函数用于过滤危险路径如../，但在将路径从Windows格式转换为Unix格式之前进行检查，导致攻击者可通过插入特定格式的符号链接绕过安全检查。在Zimbra中，用于扫描电子邮件附件的Amavis工具使用unrar处理.rar文件，攻击者可附加恶意.rar文件，将JSP shell写入公共目录，如../../../../../../../../../../opt/zimbra/jetty_base/webapps/zimbra/public/shell.jsp，从而利用此漏洞。

产品厂商： Zimbra

产品名称： UnRAR

影响版本： < 6.12

CVE-2023-30333—Zimbra-UnRAR

  1. Mô tả lỗ hổng & Mức độ nghiêm trọng
    Lỗ hổng CVE-2022-30333 ảnh hưởng tới tiện ích UnRAR < 6.12 trên hệ điều
    hành Linux/UNIX. Kẻ tấn công có thể khai thác để thực hiện path traversal, ghi
    đè hoặc ghi file tùy ý ra ngoài thư mục giải nén, dẫn đến thực thi mã tùy ý.
    • CVSS v3.1: 7.5 (High)
    • Tính chất: Không yêu cầu quyền (no privileges), không cần tương tác
    người dùng (no user interaction).
    • Tình trạng: Đã có khai thác thực tế (actively exploited) – xác nhận bởi
    NVD, CISA, CERT-EU và SonarSource.
  2. Cơ chế khai thác kỹ thuật
    Lỗi nằm trong hàm IsRelativeSymlinkSafe() của UnRAR, vốn dùng để lọc các
    đường dẫn nguy hiểm như ../. Tuy nhiên, kiểm tra này diễn ra trước khi chuyển
    đổi định dạng đường dẫn từ Windows sang Unix bằng hàm DosSlashToUnix().
    → Kẻ tấn công có thể chèn symbolic link dạng ......\tmp/shell, vượt qua kiểm
    tra an toàn, và khi giải nén trên Linux, nó sẽ trở thành ../../../tmp/shell, ghi file
    ra ngoài thư mục dự kiến.
    Trong Zimbra:
    • Công cụ Amavis (dùng để quét email đính kèm) sử dụng unrar để xử lý
    file .rar.
    • Kẻ tấn công có thể đính kèm file .rar độc hại, ghi JSP shell vào thư mục
    public
    vd: ../../../../../../../../../../opt/zimbra/jetty_base/webapps/zimbra/public/shell.jsp
    Công cụ unrar không kiểm tra đầy đủ hoặc không chuẩn hóa đường dẫn trước
    khi ghi file ra đĩa.
