Pulse Connect Secure 远程代码执行漏洞

漏洞信息

漏洞名称： Pulse Connect Secure 远程代码执行漏洞

漏洞编号：

  • CVE： CVE-2021-22893

漏洞类型： 命令执行

漏洞等级： 严重

漏洞描述： Pulse Connect Secure 9.1R11.4及以下版本存在未认证的远程代码执行漏洞。攻击者可以利用路径遍历漏洞写入任意文件，通过上传webshell导致远程代码执行。此漏洞允许攻击者在Linux系统上执行数学计算以验证RCE，或在Windows系统上执行netstat -an命令，并通过特定的正则表达式模式验证结果。

产品厂商： ivanti

产品名称： connect_secure

影响版本： <= 9.1R11.4
