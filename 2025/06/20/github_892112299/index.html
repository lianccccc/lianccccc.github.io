FTP Access Control Check

漏洞信息

漏洞名称： FTP Access Control Check

漏洞类型： 配置问题

漏洞等级： 中危

漏洞描述： FTP访问控制检查确保通过配置IP地址过滤器限制FTP访问。没有这些限制，未经授权的网络可能访问FTP服务，增加了数据泄露和其他安全威胁的风险。

产品厂商： Microsoft

产品名称： IIS FTP
