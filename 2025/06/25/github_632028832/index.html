Apache Dubbo 反序列化漏洞

漏洞信息

漏洞名称： Apache Dubbo 反序列化漏洞

漏洞编号：

  • CVE： CVE-2019-17564

漏洞类型： 反序列化

漏洞等级： 严重

漏洞描述： Apache Dubbo在启用HTTP远程调用时存在不安全的反序列化漏洞。攻击者可以提交一个包含Java对象的POST请求，完全控制Apache Dubbo的Provider实例，如果该实例启用了HTTP。此漏洞影响Apache Dubbo 2.7.0至2.7.4、2.6.0至2.6.7以及所有2.5.x版本。

产品厂商： apache

产品名称： dubbo

影响版本： 2.5.x-2.7.4

搜索语法： app=”apache-dubbo”
